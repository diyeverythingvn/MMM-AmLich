## Install
```sh
cd ~/MagicMirror/modules
git clone https://github.com/iThingscc/MMM-AmLich
```

## Configuration
```sh
nano ~/MagicMirror/config/config.js
```

```js
{
	module: "MMM-AmLich",
	position: "top_left",
	config: {
	}
},
```
